### Summary

This is a simple app with node express mongodb mongoose.
App is listening on port 3000.

#### Requirements

Node
Docker
docker-compose
typescript

##### Setup

Install package in src:

- cd src && npm install

Go to root directory:

- cd .. && docker-compose up

#### Test

Workdir: src

- cd src && npm run test

#### Debug

With vscode

- Use the launch.json
